# 重载
> 重载声明是指一个与之前已经在该作用域内声明过的函数或方法具有相同名称的声明，但是它们的参数列表和定义（实现）不相同。C++ 允许在同一作用域中的某个函数和运算符指定多个定义，分别称为函数重载和运算符重载。

> 运算符重载：重载的运算符是带有特殊名称的函数，函数名是由关键字 operator 和其后要重载的运算符符号构成的。
>
> `Box operator+(const Box&);`

#### 不可重载的运算符
> - . : 成员访问运算符
> - .*, .-> : 成员指针访问运算符
> - :: : 域运算符
> - sizeof : 长度运算符
> - ? : : 三元条件运算符
> - # : 预处理运算符
