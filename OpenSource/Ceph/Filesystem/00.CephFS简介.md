# CephFs
## 简介
> Ceph File System(CephFS)是 `POSIX` 兼容的文件系统，它建立在分布式对象存储系统RADOS之上，旨在为各种应用如，目录共享、HPC暂存空间、分布式工作流共享存储等提供最新、多用途、高可用、高性能得文件存储;如图1所示，CephFS有如下得特性：
- 文件元数据和数据分别存储于不同得数据池中；由此可以给元数据提供独立得SSD存储池，提升元数据服务性能；
- 文件系统服务由可以动态伸缩得元数据服务器MDS提供，根据系统负载可调整MDS数量；
- MDS会将元数据得操作整合成一系列得日志(Journal)存储于元数据池中，并不会在本地存储任何数据
- CephFS使用目录分片对目录树进行动态均衡到多个MDS上，避免出现热点数据；
- 客户端直连RADOS集群进行数据得读写，不需要经过gateway类似得服务；

![图1](http://qbwtbolro.bkt.clouddn.com/ceph/cephfs-architecture.PNG)
> 图1 CephFS整体架构图

CephFS有以下几种使用方式：
- Kernel Client，内核态接口，使用mout命令来挂载CephFS文件系统；
- FUSE用户空间文件系统，使用ceph-fuse命令来挂载CephFS到操作系统指定的目录；
- 使用libcephfs提供的API和CephFS进行交互；
